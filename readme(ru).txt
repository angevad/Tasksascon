Задание 2
Напишите консольное приложение, которое выводит список всех файлов и папок, полные пути которых соответствуют маске поиска ( «*» - любые символы в любом количестве, «?» - любой один символ ). Пример маски: «*Program Files\*\Bin\*.exe». Параметры запуска:
MyFind.exe -F “путь” -R “маска поиска” 
-F “Путь”. Задает папку для поиска. Если не указано — используется текущая папка. Например: «C:\Program Files»
-R Выполнять рекурсивный поиск

Например, чтобы найти все «exe» файлы, которые лежат в папке «Bin», где-то в «C:\Program Files», нужно запустить:
MyFind.exe -F “C:\Program Files” -R “*\Bin\*.exe” 
или
MyFind.exe -F “C:\Program Files” -R “*Program Files\*\Bin\*.exe”
